#  AI Knowledge Twin

An AI system that learns from your personal notes and builds a searchable knowledge base.

##  Features
- Upload PDF notes
- Extract and process content
- Convert text into embeddings
- Store knowledge using FAISS vector database
- Foundation for personalized AI assistant

##  Tech Stack
- Python
- Streamlit
- LangChain
- FAISS
- HuggingFace Embeddings

##  Demo Flow
1. Upload a PDF
2. System processes content into chunks
3. Converts into embeddings
4. Stores knowledge in vector database

##  Future Scope
- Ask questions from your notes
- Personalized learning assistant
- Weak area detection
